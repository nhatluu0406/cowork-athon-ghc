# Cowork GHC branding assets

- `cowork-ghc-logo.svg`: source vector.
- `cowork-ghc-logo-1024.png` … `16.png`: raster sizes.
- `cowork-ghc.ico`: Windows application icon.

Suggested wiring:

- Electron/electron-builder icon: `docs/assets/cowork-ghc.ico` or copy it to the existing packaging asset location.
- README/Docs: `docs/assets/cowork-ghc-logo-256.png`.
- In-app top-left mark: 24–28 px PNG/SVG while preserving the orange squircle background.

Do not stretch, recolor, add text inside the mark, or place it on a competing orange background without spacing.
